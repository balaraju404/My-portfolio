Hybrid Beamforming for IRS aided Millimeter Wave MIMO System


Overview:

Designed and implemented a Hybrid Beamforming system for a Millimeter-Wave (mmWave) Multiple Input Multiple Output (MIMO) communication system enhanced by Intelligent Reflecting Surfaces (IRS). Integrated a Greedy Algorithm to reduce the bit error rate (BER).

Key Contributions:

Hybrid Beamforming Implementation: Developed and implemented a hybrid beamforming scheme optimizing the use of both digital and analog beamforming for efficient signal transmission in mmWave frequencies. Integration of Intelligent Reflecting Surfaces (IRS): Incorporated IRS technology to enhance signal propagation, allowing for adaptive signal reflection to improve coverage and signal quality. Greedy Algorithm for BER Reduction: Implemented a Greedy Algorithm to dynamically adjust beamforming parameters, reducing bit error rates and enhancing overall system reliability.

Results:

Demonstrated significant improvement in communication range and reliability through effective beamforming techniques. Achieved enhanced spectral efficiency and increased data rates with the integration of both mmWave MIMO and IRS technologies. Successfully reduced bit error rates, improving the robustness of the communication system. Challenges and Solutions: Addressed challenges related to interference and signal attenuation in mmWave frequencies through innovative hybrid beamforming solutions. Developed and fine-tuned the Greedy Algorithm to optimize beamforming parameters for minimal bit errors.

Technologies Used:

MATLAB for simulation, algorithm development, and performance evaluation.
